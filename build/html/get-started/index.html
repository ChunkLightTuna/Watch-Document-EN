2. Download settings¶

The ESP Flash Tool setting operation is as follows:

  • COM: Burn device port selection, if there is only one device, you can choose directly.

  • BAUD: Download the baud rate, suggest 921600

  • SPI SPEED: 40MHz

  • SPI MODE: DIO

  • FLASH SIZE: 128Mbit
